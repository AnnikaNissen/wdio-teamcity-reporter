wdio-teamcity-reporter
======================

Teamcity reporter for WebdriverIO. Under development.


## Installation

```bash
npm install wdio-teamcity-reporter --save-dev
```


## Configuration

```javascript
exports.config = {
  // ...
  reporters: ['wdio-teamcity-reporter'],
  // ...
}
```
